== Welome to Monad

Monad是一个针对数据库的的全功能全文检索和分析系统，包含了自动对数据库的提取、转换、索引、存储、查询等。

Moand提供了数据增量处理、中文智能分词、分布式数据存储、并发控制优化、索引速度优化、查询优化等产品级特性。

=== Monad组成

monad分为七大模块，分别是:
    monad-cloud     云控制，负责记录所有配置文件，以及分布式协调
    monad-group     组管理服务，用来管理某一组的关于各种资源的配置
    monad-sync      数据同步服务，主要用来同步数据库到云存储中
    monad-nosql     一个KV单机/网络版本数据库
    monad-node      节点实例，负责索引的维护,以及提供搜索功能
    monad-api        API服务，提供给第三方应用的API接口服务,方便各种第三方系统接入服务
    monad-web       WEB应用程序，提供给业务人员使用，提供个性化的前端展示

提供了标准版本和分布式两个版本

=== Monad提供功能

共提供了如下功能：
    1. 全文检索，提供各种资源的统一模糊搜索，覆盖所有资源
    2. 精确搜索，提供针对某一特定资源的各个字段的检索
    3. 关联分析，针对某一具体对象（人或者物）分析出来其关联的其他信息，譬如：查询，同行、同住等关系
    4. 动态轨迹分析，全文检索所有动态资源，统一展示，某人或者某物的轨迹列表，能一眼知道某对象的动态活动轨迹
    5. 频次分析，通过针对某一动态资源进行搜索分析，分析出来对应的活动发生频次
    6. 跨资源的无目标分析，譬如查询在特定时间段，在几个地方活动的人,在数据分析中实现
